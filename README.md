# 🏠 Home Automation System

A Java Swing desktop application that simulates smart home control.

## Features
- Toggle Lights ON/OFF
- Toggle Fan ON/OFF
- Adjust AC Temperature (16°C - 30°C)
- Live Event Log with timestamps

## Technologies Used
- Java
- Java Swing (GUI)
- Eclipse IDE

## How to Run
1. Clone this repository
2. Open in Eclipse IDE
3. Run HomeAutomationSystem.java
